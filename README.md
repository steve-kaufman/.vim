# My .vim folder for linux

## Install
```bash
cd
```
SSH
```bash
git clone git@gitlab.com:tertelgames/vim.git .vim
```
HTTPS
```bash
git clone https://gitlab.com/tertelgames/vim.git .vim
```


## Setup
Move ```.vimrc``` to ```~/```
